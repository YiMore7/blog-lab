## 写作风格特点

根据「写在前面.md」：
- **只记事实** - 流水账式记录，不过度修饰
- **降低摩擦** - 不追求完美遣词造句
- **便于 AI 化处理** - 结构化事实记录
- **关注行动** - 注意力、效率、质量

## 写作文件夹结构

```
Writing/
├── 工作项目/          # 工作相关记录
├── 剪藏中转/          # 外部收集的内容
├── 周月刊/            # 客观事实记录
├── AI相关/            # AI学习和思考
├── 关系成长/          # 人际关系思考
├── 总结回顾/          # 阶段性总结
└── 周月刊洞察/        # AI生成的洞察（新）
```

## 洞察中应关注的核心矛盾

1. **收集 vs 实践** - 大量剪藏但行动不足
2. **规划 vs 执行** - 写了很多计划但落地如何？
3. **焦虑 vs 平常心** - AI心态的真实转变？
4. **关系渴望 vs 实际投入** - 说要改变但做了什么？
5. **自我认知 vs 实际行为** - 知道要做什么 vs 真正在做什么
