# 使用AI快速产出高保真原型的4种方法

为什么要快：
1. 产出效率：解放牛马，释放认知带宽做更高价值的设计与判断，而不是摸鱼
2. 质量：比大部分人画的好，带交互
3. 说的清楚：拿去给设计、老板看，便于理解一致；作为PRD的补充，给研发一个“会动的”参考，消除歧义。缩短从“想法→可视化方案”的时间，提升探索速度与方案产出密度
4. 追求：国外的产品经理都是自己出demo


## 一、说明

介绍4种通过AI工具来产出产品原型的方法，提升方案产出效率

1. 通过Agent来完成：如垂直Agent，Coze空间；通用Agent Manus等
2. 通过编程IDE来完成：如Cursor、Trae等
3. 通用大模型的Canvas功能：Gemini2.5Pro、Claude等
4. 使用设计软件来完成：如Mastergo，Figma

每个方法，我都会首先展示效果，其次演示实现的过程，最后说明此方案的利弊，过程中一些介绍一些AI能力
最后我给到一些使用场景下的建议，什么场景应该用什么方法
