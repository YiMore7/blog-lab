# 个人网站项目

## 项目概述

这是一个现代化的个人网站，采用Apple风格的设计理念，展示了个人简介、项目作品、博客文章和联系方式等核心功能。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **Markdown**: React Markdown + Remark GFM
- **状态管理**: Zustand
- **图标**: Lucide React

## 核心功能

- 个人头像展示

### 4. 联系方式

## 项目结构

```
src/
├── components/          # 通用组件
│   ├── ui/             # UI组件库
│   ├── About.tsx       # 个人简介
│   ├── Projects.tsx    # 项目展示
│   ├── Blog.tsx        # 博客系统
│   ├── Contact.tsx     # 联系方式
│   ├── Header.tsx      # 导航栏
│   └── Layout.tsx      # 页面布局
├── pages/              # 页面组件
│   └── Home.tsx        # 主页
├── hooks/              # 自定义钩子
├── lib/                # 工具函数
└── assets/             # 静态资源
```

## 使用说明

1. **本地开发**:

2. **代码检查**:
   ```bash

   ```

3. **生产构建**:
