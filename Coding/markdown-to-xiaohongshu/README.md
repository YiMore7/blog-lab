# AI Talk 图片生成器

将 AI 对话转换为精美的小红书风格图片的专业工具。

## 🌟 功能特色

### 💬 智能对话格式
- **双角色对话**：用户与AI的智能对话形式
- **Markdown 支持**：支持加粗、列表、代码块等格式
- **实时预览**：即时查看对话效果

### 🎨 精美主题设计
- **极简黑白**：经典黑白配色，简约大方
- **渐变极光**：梦幻紫色渐变，浪漫唯美
- **毛玻璃**：现代毛玻璃效果，科技感十足
- **梦幻渐变**：蓝粉渐变，清新自然

### ⚙️ 个性化定制
- **字体大小**：14-22px 可调
- **头像显示**：可选择显示/隐藏头像
- **时间戳**：自动添加日期信息
- **高清导出**：1080x1440 像素，3倍高清

### 📱 完美适配
- **小红书比例**：3:4 黄金比例（1080x1440）
- **高清画质**：支持3倍像素比导出
- **一键下载**：PNG格式，方便分享

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📖 使用说明

### 1. 输入对话内容
使用以下格式输入对话：
```
**用户**: 你好，我想学习编程，有什么建议吗？

**AI**: 你好！学习编程是个很棒的选择。我建议从Python开始，它语法简单，适合初学者。

**用户**: 好的，我该怎么开始学习呢？

**AI**: 你可以按照以下步骤：
1. 选择一个入门课程
2. 安装Python环境
3. 动手写第一个程序
4. 坚持练习，每天进步一点点
```

### 2. 选择主题样式
- 极简黑白：适合正式场合
- 渐变极光：适合分享生活感悟
- 毛玻璃：适合科技话题
- 梦幻渐变：适合轻松话题

### 3. 调整显示选项
- 字体大小：根据内容多少调整
- 显示头像：增加对话的真实感
- 显示时间戳：让内容更完整

### 4. 导出图片
点击"下载高清图片"按钮，即可获得精美的小红书风格对话图片。

## 🎯 使用场景

- **学习分享**：分享AI学习心得
- **技术讨论**：展示AI对话过程
- **生活感悟**：记录有趣的AI对话
- **知识科普**：传播AI相关知识

## 🛠️ 技术栈

- **React 18** + **TypeScript**：现代化前端开发
- **Tailwind CSS**：原子化CSS框架
- **html-to-image**：高质量图片导出
- **react-markdown**：Markdown渲染引擎
- **lucide-react**：精美图标库
